Zendesk JWT Auth Setup

Configure your Zendesk account — works with any messaging widget

🔒 Privacy: Your credentials are stored in your browser's sessionStorage only — they are never saved to any server or disk. They disappear automatically when you close this browser tab.

📍 How to find your Zendesk Conversations API key

A Conversations API key has three pieces: App ID (your account identifier), Key ID (included in your JWT header as kid), and Secret key (used to sign your JWT).

  1. In Zendesk Admin Center, go to Apps and integrations → APIs → Conversations API
    Requires Zendesk Suite Professional plan or above
  2. Click "Add key" (or "Create new key")
  3. Enter a descriptive name (e.g. jwt-auth-app) and click Create
  4. In the dialog that appears, copy the values:
    • Key ID — looks like app_67097c94d8a020e6a236ae87
    • Secret key — a long random string
  5. For the Widget Key, go to Channels → Messaging and Social → Messaging, click on your messaging widget link, then go to Installation. Copy the key= value from the snippet:
    <!-- The widget key is the value after key= -->
    <script id="ze-snippet"
      src="https://static.zdassets.com/ekr/snippet.js?key=your_widget_key_here">
    </script>
⚠️ Important: The secret key is only shown once when you create it. If you lose it, you'll need to create a new key. For more details, see the Zendesk Conversations API keys documentation.
